BIN 可执行文件 启动停止的bat
conf 配置文件 xml
lib jar文件
webapps 可执行的项目 我们开发的项目
work jsp编译成的java文件+class文件

jsp–>java–>calss

webapps是项目默认的虚拟路径
可以手动修改 server.xml 的appbase
同级下增加<context docBase=”” path= -可以写web apps以下的相对路径-/>
需要重启

或者在
—/tomcat\6\conf\Catalina\localhost中新建xml文件输入
<context docBase=”” path= -可以写web apps以下的相对路径-/>
不需要重启
